# LevelUp
# ⚡ XP Goal Tracker with Daily Motivation

##  About the Project

This project is a simple web app where users can set their daily goals and track their progress. The main idea is to make productivity a bit more fun by using an XP (experience points) system. When tasks are completed, the user gains XP, and if tasks are not completed, some XP is deducted.

Along with this, the app also shows a random motivational quote every time, so users can stay inspired while working on their goals.

---

##  Objective

The goal of this project is to practice and understand:

* How JavaScript works in real applications
* How to use APIs using `fetch()`
* How to use functions like `map`, `filter`, and `reduce`
* How to build a clean and interactive UI

---

##  APIs Used

* **Quotable API**
  https://api.quotable.io/random

* **Type.fit Quotes API (Backup)**
  https://type.fit/api/quotes

These APIs are used to display motivational quotes in the app.

---

##  Features

* Add daily tasks
* Mark tasks as completed
* Track completed and pending tasks
* Gain XP for completing tasks
* Lose XP for incomplete tasks
* Extra penalty if less than half of the tasks are completed
* Level system based on XP
* Display motivational quotes

---

##  Concepts Used

* Fetch API for getting data
* Array methods:

  * `map()` to display tasks
  * `filter()` to separate completed/pending tasks
  * `reduce()` to calculate total XP
  * `sort()` to arrange tasks
* DOM manipulation
