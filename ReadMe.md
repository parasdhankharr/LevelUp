📰 News Search App

A simple and responsive News Search App that fetches real-time news using the GNews API. Users can search for any topic and get the latest articles instantly.

Features:->

    Search news by keyword
    Press Enter to fetch results
    Fast API fetching using fetch()
    Error handling (empty input, API errors, no results)
    Clean and simple UI


Tech Stack :->

    HTML
    CSS
    JavaScript (Vanilla JS)
    GNews API

How It Works:->

    User types a query in the search box
    Presses Enter
    App sends request to GNews API
    News articles are displayed dynamically

API Setup:->

    Go to: https://gnews.io/
    Sign up and get your API key
    Replace this in your code:
    const apiKey = "YOUR_API_KEY_HERE";
